# Astral Party - Game Research Data

Private repository containing game vulnerability research, GM interfaces, and knowledge base.

## Contents
- `gm_exploits.json` - GM/Cheat interfaces (CheatItem, DevCharge, GMPlayerSetting)
- `vulnerability_report.json` - Full vulnerability scan results
- `singleton_discovery.json` - Memory offsets and singleton patterns
- `game_addresses.json` - Network architecture and RPC protocol
- `button_analysis.json` - UI button classification (RPC-safe vs dangerous)
- `wiki_bilibili/` - Game data (characters, cards, chips, maps, mechanics)
